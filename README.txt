============================================
Video To MP3 Converter
============================================

Description
-----------
This software automatically converts video files to MP3 audio files.

How to Use
----------
1. Initial Setup:
   - We recommend moving this program to a dedicated folder (not just leaving it in your Downloads folder)
   - When you first run the program, it will automatically create two folders:
     * "input-videos" folder - For your video files
     * "output-audios" folder - Where converted audio files will appear

2. Normal Use:
   - Place your video files in the "input-videos" folder
   - Run the program by double-clicking on "Video To MP3.exe"
   - Find your converted MP3 files in the "output-audios" folder

3. The software will convert all video files in the input folder to MP3 audio files.

4. Important: After conversion, delete the video files you no longer need from the "input-videos" folder. 
   The program converts all videos in the folder each time it runs, so keeping old files will increase conversion time.

Supported File Types
-------------------
The software supports the following video file types:
- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)

Notes
-----
- If the input or output folders don't exist, the software will create them automatically.
- The file includes a lightweight version of FFmpeg, so no additional installation is required.
- This program is portable and does not "install" itself on your computer. It simply runs when you execute it.
- To remove the program, just delete the EXE file and its folders.
- We recommend creating a dedicated folder for this program (such as "Audio Extractor" on your desktop) for easy access.
